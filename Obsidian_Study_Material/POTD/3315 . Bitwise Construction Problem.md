

### Problem:

Find minimum `x` such that:

`x | (x + 1) == nums[i]`

---

### 🔑 Core Observations

1. `x | (x + 1)` is **always odd**
    
2. If `nums[i]` is even → **impossible**
    
3. If `nums[i]` is odd → solution exists
    

---

### 🔹 Key Rule (Final Truth)

> **Minimum `x` is obtained by clearing the lowest zero bit of `nums[i]`**

---

### 🔍 Find Lowest Zero Bit

`int positionOfFirstZeroBit(int num) {     int pos = 0;     while (num & 1) {         num >>= 1;         pos++;     }     return pos; }`

Or one-liner:

`int pos = __builtin_ctz(~num);`

---

### 🔹 Construct Answer

`if (nums[i] is even) → -1 else:     b = lowest zero bit position     ans = nums[i] - (1 << b)`

---

### 🧪 Examples

|nums|binary|b|ans|
|---|---|---|---|
|3|11|2|1|
|5|101|1|4|
|7|111|3|3|
|11|1011|2|9|
|13|1101|1|12|
|2|10|—|-1|
