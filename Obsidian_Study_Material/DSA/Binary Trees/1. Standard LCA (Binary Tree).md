### Intuition: "The Split Point"

The LCA is the lowest node (deepest) that has both `p` and `q` as descendants.
Think of it as finding the **junction** where the paths to `p` and `q` diverge.

- **Case 1 (Divergence):** If `p` is in the Left subtree and `q` is in the Right subtree, the current node **IS** the LCA. It is the split point.

- **Case 2 (One Side):** If both `p` and `q` are in the Left subtree, the LCA must also be in the Left subtree (so we ignore the Right).

- **Case 3 (Direct Ancestry):** If the current node is `p`, and `q` is somewhere below it, then `p` itself is the LCA.

### The Algorithm (Recursive DFS)

We use a **Bottom-Up** approach (conceptually). We ask our children: "Did you find `p` or `q`?"

**The Logic Flow:**
1. **Base Case:**
    - If I am `null`, return `null`.
    - If I am `p` or `q`, return **myself**. (I found one of them!).

2. **Recursive Step:** Ask Left and Right children.

3. **Decision:**
    - If **both** Left and Right return a node (non-null), it means one target is on the left and one is on the right. **I am the LCA.** Return myself.
    - If only **Left** returns a node, the LCA (or both targets) is down the left side. Return the Left result.
    - If only **Right** returns a node, the LCA is down the right side. Return the Right result.

```cpp
TreeNode* LowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
	// Base Cases should be handled
	if(!root || root == p || root == q) return root;
	
	// Ask to the Childs : recursively
	TreeNode* left = LowestCommonAncestor(root->left, p, q);
	TreeNode* right = LowestCommonAncestor(root->right, p, q);
	
	// If both left and right are non-null then return the root
	if(left && right) return root;
	
	return left ? left : right; // If one of them is null then move towards the non-null part
}
```
