You are **not given the nodes** whose LCA you want.

Instead:
- The nodes are defined **implicitly**
- “Deepest” means **maximum depth from root**
- There may be:
    - one deepest node
    - many deepest nodes
    - deepest nodes on both left & right subtrees

Code
```cpp
pair<int, TreeNode*> getLCA(TreeNode* root) {
	// Assign zero depth for nullptr nodes, so leaf node depth become 1
	if(!root) return {0, nullptr};
	
	auto left = getLCA(root->left);
	auto right = getLCA(root->right);
	
	if(left.first == right.first) {
		// When depth matches return the root cause this will be the LCA 
		return {left.first + 1, root};
	}
	else if(left.first > right.first) {
		return {left.first + 1, left.second};
	}
	else {
		return {right.first + 1, right.second};
	}
	
	return {0, nullptr};
}

TreeNode* subtreeWithAllDeepest(TreeNode* root) {
	return getLCA(root).second;
}
```
